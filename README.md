# 💰 Finance Toolkit

A modern collection of financial calculators built with **React**,
**Vite**, **TypeScript**, and **Tailwind CSS**.

## ✨ Features

### 🏷️ Discount Calculator

-   Calculate discounts
-   Tax-inclusive/exclusive support

### 👥 Expense Split Calculator

-   Split expenses
-   Calculate settlements
-   Local Storage

### 🏦 Loan Repayment Calculator

-   Loan amount
-   Loan term
-   Annual interest
-   Repayment schedule
-   Export to Excel

## 🛠️ Tech Stack

-   React
-   Vite
-   TypeScript
-   Tailwind CSS
-   React Router DOM
-   React Hook Form
-   Yup
-   Lucide React
-   SheetJS (xlsx)

## 📦 Installation

``` bash
npm install
npm run dev
```
